# handy.profile for filejail
A firejail profile for handy (https://handy.computer), first written
03/18/2026 by Lester Hightower and Google Gemini. Place handy.profile
at path ~/.config/firejail/handy.profile and hosts.handy into /etc/.

## Purpose
My purpose in writing this was 100% cybersecurity related. I have no
reason to suspect that handy will try to do anything bad, but I also
have no reason to trust it.

## Audio
Note that this was written on Mint Linux 21.1 which uses pulseaudio,
and so this profile may need some slight alterations for other audio
subsystems.
